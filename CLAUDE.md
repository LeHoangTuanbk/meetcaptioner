# MeetCaptioner

Easily capture your meeting's caption at Google Meet.
Easily translate the live caption to any languages using LLMs.
--
Teck Stacks:

- pnpm
- WXT Next-gen Web Extension Framework
- React, Typescript
- CSS: Tailwind
- UI library components: shadcn

## Important Notes

You MUST access the coding convention files in documents/development/coding-convention/ before writing code
Follow the existing patterns in the codebase
Maintain consistency with project standards
Before installing anything, remember to check package.json to select the most appropriate command.
